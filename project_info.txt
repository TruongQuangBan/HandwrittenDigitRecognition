2. Chuc nang chinh cua tung file/thu muc
----------------------------------------
- app.py:
  Ung dung Flask phuc vu giao dien web va API du doan chu so. File nay tai model.pkl bang joblib, lay StandardScaler va LogisticRegression da huan luyen, sau do cung cap cac route:
  + GET /: tra ve static/index.html.
  + POST /predict: nhan mang 784 gia tri pixel, chuan hoa bang scaler va du doan chu so.
  + POST /predict_base64: nhan anh base64, giai ma bang Pillow, chuyen ve anh grayscale 28x28, flatten thanh 784 gia tri va du doan.

- train.py:
  Script huan luyen mo hinh nhan dien chu so viet tay tren tap MNIST. File nay tai du lieu MNIST tu cache cuc bo mnist_784.pkl neu co, neu chua co thi tai tu OpenML. Sau do chia 60.000 anh train va 10.000 anh test, luu anh mau vao thu muc images/, chuan hoa du lieu bang StandardScaler, huan luyen LogisticRegression one-vs-rest, danh gia accuracy tren tap test, luu model.pkl va ghi report.txt.

- static/index.html:
  Giao dien web client gom HTML, CSS va JavaScript. Nguoi dung ve chu so tren canvas 280x280, JavaScript thu nho anh ve 28x28, chuyen thanh mang 784 gia tri grayscale, gui len API /predict bang fetch va hien thi ket qua du doan.

- Dockerfile:
  Dinh nghia Docker image cho ung dung. Image dung python:3.9-slim, cai thu vien tu requirements.txt, copy ma nguon vao /app, expose cong 5000 va chay python app.py.

- .dockerignore:
  File cau hinh cac tep/thu muc khong nen dua vao Docker build context neu da duoc khai bao.

- requirements.txt:
  Danh sach thu vien Python can cai dat: flask, scikit-learn, numpy, pillow, joblib.

- report.txt:
  Bao cao ket qua huan luyen gan nhat, gom so luong anh train/test, thoi gian huan luyen va do chinh xac tren tap test.

- model.pkl:
  File model da huan luyen, gom StandardScaler va LogisticRegression duoc luu bang joblib. app.py dung file nay de du doan.

- mnist_784.pkl:
  Cache cuc bo cua tap du lieu MNIST gom data va target, giup train.py khong phai tai lai tu OpenML trong nhung lan chay sau.

- images/:
  Thu muc anh minh hoa sinh ra trong qua trinh huan luyen, gom 5 anh mau tu tap train va mot anh montage ghep cac mau.

- openml_cache/:
  Thu muc cache cua scikit-learn/OpenML khi tai du lieu MNIST.

- outline.txt:
  Dan y chi tiet cho bao cao Word, gom gioi thieu, tong quan he thong, co so ly thuyet, xay dung he thong, trien khai Docker & Cloud, ket qua danh gia, ket luan va tai lieu tham khao.

- project_info.txt:
  File tong hop thong tin du an hien tai, cau truc thu muc, chuc nang file, cong nghe su dung, link GitHub va so lieu huan luyen.

- .DS_Store:
  File an do macOS tao tu dong, khong phai thanh phan logic cua ung dung.

3. Thu vien va cong nghe su dung
--------------------------------
- Python 3.9: ngon ngu chinh cho backend va script huan luyen.
- Flask: framework web nhe de phuc vu giao dien va API REST.
- scikit-learn: tai du lieu MNIST, chuan hoa du lieu, huan luyen LogisticRegression va tinh accuracy.
- LogisticRegression one-vs-rest: mo hinh phan loai 10 lop chu so 0-9 bang chien luoc one-vs-rest.
- StandardScaler: chuan hoa dac trung pixel truoc khi huan luyen va du doan.
- NumPy: xu ly mang pixel va du lieu dau vao cho model.
- Pillow: doc, giai ma va xu ly anh base64 trong API /predict_base64; tao anh mau MNIST trong train.py.
- joblib: luu/tai cache du lieu MNIST va model.pkl.
- HTML/CSS/JavaScript: xay dung giao dien ve chu so bang Canvas va goi API bang fetch.
- Docker: dong goi ung dung Flask thanh container de trien khai.
- OpenML/MNIST: nguon du lieu chu so viet tay 28x28.
